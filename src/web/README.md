# Web Development

## Prerequisites
- Bun installed.

## Install Dependencies
```powershell
bun install
```

## Run Vite HMR
```powershell
bun run dev
```

The dev server runs on `http://127.0.0.1:5173` and proxies `/api/*` to the Rust API at `http://127.0.0.1:8080`.

## Build
```powershell
bun run build
```
